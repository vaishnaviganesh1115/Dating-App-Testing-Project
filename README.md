# Dating-App-Testing-Project

**Project Description:**
  This project focuses on performing Front-end manual testing of a sample Dating App (UI-only prototype with no backend integration)
  he aim is to identify defects related to:
      UI/UX flow
      screen navigation
      swiping functionality
      input field validations
      error handling
      responsiveness
      profile details display
  The goal is to ensure that the app’s frontend workflow functions smoothly across all screens and user actions, even without backend connectivity.

**Project Objectives:**
  Verify the smooth flow of onboarding screens
  Validate basic input behavior such as text entry, navigation, and field responses.
  Check UI consistency for elements like buttons, labels, icons, profile cards, and images.
  Test swiping functionality (left → reject, right → like) across multiple profiles.
  Validate the profile details page (Name, Age, About, Hobbies, Favorites).
  Identify and report defects in a structured and clear format.

**Types of Testing Performed:**
  Functional Testing
  UI/UX Testing
  Usability Testing
  Negative Testing
  Validation & Error Message Testing

**Tools & Technologies Used**
  Documentation: Microsoft Excel, Microsoft Word
  Test Planning: Excel (Test scenerios & Test cases)
  Bug Tracking: Excel Bug Report
